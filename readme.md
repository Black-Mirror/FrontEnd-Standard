## 细节规范
  - 文件名如无特殊必要，须使用小写
  - 文件名要么以命名空间为前缀，如 `ruleset-list.html`, `ruleset-update.html`; 要么分子目录
  - js, css, html 如无特殊必要，须使用 2 个空格缩进
  - 代码中有大段的注释要写明注释的原因，比如 js 某段注释写上“此方法暂时没实现好”
  - js, css, html 等源文件统一的 todo 前缀：`TODO:` 方便全局搜索查找
